# Advent of Code 2022

Please see the [Advent of Code website](https://adventofcode.com/) for more information.

## Install Elixir

See [Installing Elixir](https://elixir-lang.org/install.html).

## Run a Puzzle Script

```bash
elixir <PUZZLE FILE>
```

> To run interactively, use `iex` instead of `elixir`.
